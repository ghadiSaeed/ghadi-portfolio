InsightMind AI

AI-Powered Research Assistant for Mental Health Literature designed to help researchers analyze academic papers efficiently using Large Language Models.

GitHub Repository Live Demo Portfolio

Project Overview

InsightMind AI is a dedicated tool built to accelerate academic research. By leveraging modern Large Language Models, it allows users to upload and analyze multiple PDF research papers locally in the browser, extract automated insights, generate interactive mind maps, and create presentations directly from the document contents.

The application runs entirely in the browser, providing a seamless and private experience for interacting with complex academic literature.

Problem Statement

Researchers often spend significant time reviewing lengthy academic papers and manually organizing findings. InsightMind AI was created to streamline this workflow by combining document analysis, AI-assisted questioning, and automated content generation into a single browser-based application.

Key Features

  • PDF Management: Upload and analyze multiple PDF research papers.
  • PDF Viewer: Built-in viewer with page navigation and text search.
  • AI Question Answering: AI-powered document querying.
  • Research Dashboard: Automatic topic extraction and visual insights.
  • Trend Analysis: Keyword and research trend identification.
  • Literature Analysis: Generation of Literature Reviews and Research Gaps.
  • Interactive Mind Map: Visual representation of document concepts.
  • Presentation Generator: Automatically generates slides from research.
  • Notes & Bookmarks: Save progress and key findings.
  • Export Functionality: Export notes and presentations directly.

Technologies Used

  • HTML5 & CSS3
  • Vanilla JavaScript
  • PDF.js
  • Google Gemini 2.5 Flash API
  • Vis Network
  • html2pdf.js
  • Marked.js

AI Model Architecture

This project leverages Google Gemini 2.5 Flash as the core Large Language Model (LLM). The API is integrated directly into the browser application to process text extracted from PDFs to power the following features:

  • Research question answering
  • Topic extraction
  • Literature analysis
  • Mind map generation
  • Presentation generation
  • Research dashboard
